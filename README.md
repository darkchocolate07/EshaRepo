# Esha Nakka — FRIENDS-Themed Portfolio

A premium, playful-but-credible portfolio website for **Esha Nakka**, themed around the iconic *FRIENDS* Central Perk aesthetic.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy to [Vercel](https://vercel.com) by connecting the GitHub repository. Set `NEXT_PUBLIC_SITE_URL` to your production domain.

## Project Structure

- `app/` — Pages and routing
- `components/` — UI, layout, and section components
- `content/` — Typed content data from Context.md
- `public/` — Images, CV, and SVG assets
